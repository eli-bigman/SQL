# DEM03_lab_Inventoryand-Order-Management-System

## Overview
This project contains SQL scripts for an e-commerce inventory and order management system database.

## Contents
- **SQL_DDL.sql** - Data Definition Language scripts for creating database schema (tables, constraints, relationships)
- **SQL_DML.sql** - Data Manipulation Language scripts for inserting, updating, and querying data

## Database Structure
The system manages:
- Customer information
- Product catalog
- Inventory tracking
- Order processing
- Order items and details

## Usage
1. Run the DDL script first to create the database schema
2. Execute the DML script to populate and manipulate data
